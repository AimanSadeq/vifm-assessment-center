# 04 — Competency Rating Form (Importance / Frequency / Essential)

**Purpose:** Quantitative relevance evidence per competency per role, plus the
**Lawshe essential ratings** used to compute the Content Validity Ratio (CVR) in
Phase 3.

**Duration:** 10–15 min · **Rater:** each SME, independently.

---

## Rating form

For **each** competency, rate this role:

- **Importance** — how important to effective performance? (1 = not important …
  5 = critical)
- **Frequency** — how often is it needed? (1 = rarely … 5 = constantly)
- **Lawshe rating** — is this competency: **Essential** / **Useful but not
  essential** / **Not necessary** for the role?

| Competency | Importance (1–5) | Frequency (1–5) | Essential / Useful / Not necessary |
|---|:---:|:---:|:---:|
| Forward Strategy Setting | | | |
| Commercial & Market Awareness | | | |
| *(…all 38, plus any additions from card-sort…)* | | | |

---

## Computing the Content Validity Ratio (CVR)

For each competency, using **N** = number of SMEs who rated it and **nₑ** =
number who rated it **"Essential"**:

```
CVR = (nₑ − N/2) / (N/2)
```

- CVR ranges from **−1 to +1**. Higher = stronger agreement that the competency
  is essential.
- A competency is retained on content-validity grounds when its CVR meets or
  exceeds the **critical value** for the panel size (Lawshe, one-tailed, p = .05):

| Panel size (N) | Min CVR |
|---:|:---:|
| 8 | 0.75 |
| 9 | 0.78 |
| 10 | 0.62 |
| 11 | 0.59 |
| 12 | 0.56 |
| 13 | 0.54 |
| 14 | 0.51 |
| 15 | 0.49 |
| 20 | 0.42 |
| 25 | 0.37 |
| 30 | 0.33 |
| 40 | 0.29 |

*(Values per Lawshe, 1975 / Wilson et al., 2012 revision. Use the row matching
your actual panel size.)*

- Also compute the **Content Validity Index (CVI)** = mean CVR across retained
  competencies, as an overall index for the validity report.

---

## How ratings combine with the other instruments

- **Importance/Frequency means** feed the README §5 include/drop thresholds.
- **CVR** is the formal content-validity statistic for the Phase 3 report.
- Where the rating form and the card-sort disagree, treat it as a flag to review
  the competency's wording — often a clarity problem, not a relevance problem.
